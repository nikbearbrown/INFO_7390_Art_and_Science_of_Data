    "### 1. Dataset Generation\n",
    "\n",
    "#### a) Generate Base Responses\n",
    "\n",
    "```bash\n",

    "    --output_dir \"data/judge_dataset\"\n",
    "```\n",
    "\n",
    "#### c) Create PairRM-Based Dataset\n",

    "\n",
